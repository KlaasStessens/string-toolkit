# @nikleos/string-toolkit

A javascript library for strings. 

## Explanation
The user inputs a string. The program converts all spaces in the string into underscores, and all letters become lowercase.

## Installation

### npm
```bash
npm install @nikleos/string-toolkit
```
### yarn
```bash
yarn add @nikleos/string-toolkit
```
## Testing
npm test
## Function(s)
### toSnakeCase
Converts a string into *snake_case*
```javascript
toSnakeCase("Hello World");
// result -> hello_world
```
### toKebabCase
Converts a string into *kebab_case*
```javascript
toKebabCase("Hello World");
// result -> hello-world
```
